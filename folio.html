01 / Healthcare
EMR Dashboard Redesign
Doctors struggled navigating fragmented patient data scattered across multiple disconnected screens.
The redesigned dashboard unified clinical data — vitals, lab results, medications — into a single interface with intelligent hierarchy. Information prioritized by clinical urgency, not alphabetical order.

02 / Enterprise
Document Exchange Workflow
Organizations had zero visibility into where documents were in the approval chain, creating costly delays.
Introduced document status tracking, version history, and centralized management across approval stages. Every stakeholder could see exact document state in real time.

03 / Hardware
Scanner Hardware UX
The scanning workflow required excessive interaction steps, causing repeated errors for non-technical users.
Redesigned the flow as three clear, linear stages: capture → preview → confirm. Each stage gives users full confidence before proceeding, eliminating the need to rescan.
7 → 3 steps ↓ Rescan rate Non-technical users
